# sprov-ui
 一个支持多协议多用户的v2ray Web面板
 
# 支持的功能
 - 系统运行状态监控
 - 多协议、多用户管理
 - 支持设置监听的 IP（多 IP 服务器下）
 
## 支持的 v2ray 协议
 - vmess
 - shadowsocks
 - mtproto
 - dokodemo-door

## 支持的 vmess 传输配置
 - tcp
 - kcp + 伪装
 - ws + 伪装 + tls

# 支持的系统
>务必使用纯净版的系统，建议在 256MB 内存及以上的 vps 搭建，低内存情况下可能运作不良
 - CentOS 7（推荐）
 - Ubuntu 16
 - Ubuntu 18
 - Debian 8
 - Debian 9

# 一键安装&升级面板
>面板已内置升级功能（每30分钟从 Github 检测一次）
```
wget -O install.sh -N --no-check-certificate https://github.com/ugvf2009/sprov-ui/blob/master/install.sh && bash install.sh
```
